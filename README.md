Getting Started

 Prerequisites
- Node.js 16+ installed
- npm or yarn

 Installation

```bash
 1. Navigate into the folder
cd finsire-clone

 2. Install dependencies
npm install

# 3. Start the dev server
npm start
```

The app will open at http://localhost:3000

 Build for Production

```bash
npm run build
```
